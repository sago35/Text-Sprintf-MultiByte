# NAME

Text::Sprintf::MultiByte - sprintf with multibyte chars

# SYNOPSIS

    use Text::Sprintf::MultiByte;

    spf "<%3s>", "„ÅÇ"; # multibyte char works good

# DESCRIPTION

Text::Sprintf::MultiByte is sprintf with multibyte chars.
